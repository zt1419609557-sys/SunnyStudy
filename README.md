# Codex 编程学院

一个面向小白的中文编程学习网站原型，目标是把前端、后端、数据库、部署、发布、域名这些内容讲清楚，让用户可以按需选择文章或课程学习。

## 这个仓库里有什么

- `outputs/codex-learning-site/index.html`：网站入口
- `outputs/codex-learning-site/styles.css`：页面样式
- `outputs/codex-learning-site/app.js`：课程内容、筛选、进度和交互
- `outputs/codex-learning-site/README.md`：站点内部说明
- `LICENSE`：开源协议

## 怎么打开

直接双击：

`outputs/codex-learning-site/index.html`

或者用任意静态服务器打开 `outputs/codex-learning-site` 这个目录。

## 功能

- 学习路线
- 课程库筛选
- 课程详情页
- 学习进度记录
- Codex 提示词库
- 域名和部署入门

## 适合谁

- 完全零基础的新手
- 想系统补前后端知识的人
- 想了解网站如何部署上线的人
